# 🌦️ Real-Time Weather Dashboard

An interactive dashboard that fetches and displays live weather data from global cities using external APIs.

## 🚀 Features
* **API Integration:** Fetches real-time data from OpenWeatherMap API (or equivalent).
* **Search Functionality:** Get weather details for any city worldwide.
* **Dynamic Backgrounds:** UI changes based on weather conditions (Clear, Clouds, Rain, etc.).

## 💻 Tech Stack
* **JavaScript (Async/Await & Fetch API)**
* **CSS3 Flexbox/Grid**
* **HTML5 Canvas/Icons**
